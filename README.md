# Liverpool FC Fan Site

A small, unofficial fan site about Liverpool Football Club, built with plain
HTML and CSS as a learning project at the UCD Professional Diploma in Full Stack Software Development.

## Live site

Hosted with GitHub Pages: `https://<your-username>.github.io/liverpool-fc/`

> Replace `<your-username>` with your GitHub username once Pages is enabled.

## Pages

- Home (`index.html`) — club introduction, quick facts and the anthem
- History (`history.html`) — key moments and a table of major honours
- Squad (`squad.html`) — positions guide and a selection of club legends

## Built with

- Semantic HTML5 (`header`, `nav`, `main`, `article`, `section`, `footer`)
- CSS for layout, typography and colour.

## Running locally

Open `index.html` in any web browser.

## Disclaimer

This is an unofficial fan site made for educational purposes and is **not
affiliated with Liverpool Football Club**. The crest shown is a stylised
placeholder, not the club's official badge.
